# Bundled Instrument Files

This folder is packaged and installed with the app by both `PKGBUILD` files.
By default, build/install flows fetch two high-quality SFZ packs from FreePats:
- `SalamanderGrandPiano-V3+20200602.sfz` (default acoustic grand piano, CC-BY-3.0)
- `EGuitarFSBS-bridge-clean-20220911.sfz` (default clean electric guitar, CC0)

Before adding any `.sf2` or `.sfz` here, verify redistribution rights and keep license
or attribution details with the project.

Recommended policy:
- Only include instrument packs with explicit redistribution permission.
- Add the source URL and license name for each bundled file.
- Remove any instrument pack immediately if licensing is unclear.
